### GRF Browser Implementation (Stage 1)
- **PR #16** - feat(tools): add GRF Browser GUI tool (Stage 1)

#### Features Implemented:
- Dear ImGui window with SDL2 backend (cimgui-go)
- Two-panel layout: Files tree (left) + Preview panel (right)
- File tree with virtual folder structure from GRF paths
- Live search filtering by filename
- Type filter checkboxes:
  - Sprites (.spr)
  - Animations (.act)
  - Textures (.bmp, .tga, .jpg, .png, .imf)
  - Models (.rsm)
  - Maps (.rsw, .gat, .gnd)
  - Audio (.wav, .mp3)
  - Other
- Text-based file type icons: `[SPR]`, `[IMG]`, `[MAP]`, `[GAT]`, `[GND]`, `[TXT]`
- Status bar with total/filtered file counts
- Keyboard shortcuts:
  - `Ctrl+C` - copy filename only
  - `Cmd+Ctrl+C` - copy full path (macOS friendly)
- EUC-KR to UTF-8 filename conversion for Korean support

#### Technical Changes:
- Added cimgui-go dependency (requires Go 1.24+)
- Updated CI workflow to Go 1.24
- Added grfbrowser to `make build-tools`
- Added `imgui.ini` to .gitignore

### Usage
```bash
make build-tools
./build/grfbrowser -grf path/to/archive.grf
```
